# Mojito analysis  
Code for matching `fastlisaresponse` to the Mojito simulation. 

Notebooks are not cleaned up: probably unreadable for anyone else. The main notebook I would point people to is the one on source 6: `FLR_EMRI_source_6.ipynb`, as it is best matching. Note that the timings must be adjusted, at the moment the best match is found by time-sliding one simulation against the other.
